My Tech Goals for the Next 2 Years

To achieve my goal of starting a full-stack web development agency called "Codebanju" in 18 months and be counted among Africas top 10 software developer/programmer. Breakdown of goals and skills I should focus on each quarter over the 2 years/24-month period:

First Quarter (Months 1-3):

Skills to Learn:

1. HTML (HyperText Markup Language) 2. CSS (Cascading Style Sheets) 3. JavaScript (including ES6 features) 4. PHP (Hypertext Preprocessor)

Activities:

1. Complete online tutorials and courses on HTML, CSS, JavaScript, and PHP. 2. Build simple projects to practice my skills. 3. Create a personal portfolio website using these technologies.

Second Quarter (Months 4-6):

Skills to Learn:

1. Git and Version Control 2. Responsive Web Design 3. Frontend Frameworks (e.g., Bootstrap, Vue.js, React)

Activities:

1. Learn how to use Git for version control. 2. Study responsive web design principles. 3. Build projects with Bootstrap and start learning a frontend framework like Vue.js or React.

Third Quarter (Months 7-9):

Skills to Learn:

1. Backend Development with Node.js or Django 2. RESTful API design 3. Databases (SQL, MySQL, or MongoDB)

Activities

1. Dive into backend development with Node.js or Django. 2. Design and implement RESTful APIs for your projects. 3. Learn how to work with databases like SQL, MySQL, or MongoDB.

Fourth Quarter (Months 10-12):

Skills to Learn:

1. Web Security Basics 2. Testing (Unit Testing, Integration Testing) 3. Deployment Basics

Activities:

1. Understand web security fundamentals and implement best practices. 2. Start writing tests for my projects (unit testing, integration testing). 3. Learn how to deploy web applications to servers or cloud platforms.

Fifth Quarter (Months 13-15):

Skills to Learn:

1. Advanced JavaScript Concepts (Async/Await, Promises, Closures) 2. Frontend Frameworks (Advanced) 3. Advanced Database Concepts

Activities:

1. Master advanced JavaScript concepts. 2. Deepen my knowledge of a frontend framework. 3. Explore advanced database concepts and optimization techniques.

Sixth Quarter (Months 16-18):

Skills to Learn:

1. DevOps Basics (CI/CD, Docker) 2. Project Management 3. Soft Skills (Communication, Problem-Solving)

Activities:

1. Set up CI/CD pipelines using tools like Jenkins or GitHub Actions. 2. Learn project management methodologies and tools. 3. Improve my soft skills through communication, teamwork, and problem-solving.

Seventh Quarter (Months 19-21):

Activities:

1. Build a strong portfolio of projects showcasing your skills. 2. Develop a business plan for Codebanju, including services, pricing, and target market. 3. Register my business, set up a website, and start marketing professional services.

Eighth Quarter (Months 22-24):

1. Launch "Codebanju Institute of Marketing and Technology (CIMT)" and start taking on clients. 2. Continue learning and improving my skills based on client feedback and project requirements. 3. Network with other professionals in the industry and seek mentorship opportunities to further.
